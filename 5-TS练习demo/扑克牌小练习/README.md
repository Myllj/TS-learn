# 扑克牌案例练习

## 背景

加强对TS的熟练，练习举例了扑克牌的小例子，主要练习TS的基本类型、枚举、模块化、接口、类等；



## 文件介绍

1. index.ts：TS基本类型来定义
2. index2.ts：使用枚举优化了index.ts
3. index3.ts：拆分了enums枚举，拆分了types，拆分了untils公共文件，使用模块化导入分别导入它们
